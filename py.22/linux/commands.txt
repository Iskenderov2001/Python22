CTRL+ALT+T - Команда открытия терминала
MacOS (В спотлайт вводим Терминал)

~ - Символ домашней директории(папки) 
/ - Символ корневой директории

.. - Переход на предудущую директорию
cd /home/users -> аналогичный способ 
. - Обозночает текущую директорию

1. ls(list) -> Выводит список всех директории и файлов в текущей директории(рабочая директория)

2. cd(change directory) -> меняет текущую директорию на указанную
cd <Название директории>

3. mkdir(make directory) -> Команда которая создает директорию (New folder)
mkdir <директория>
mkdir{1,2,3,4,5,6} - Создание многон кол-во директории

6. rmdir (remove directory) -> удаление директории.

4. touch -> создает указанный файл
touch <name_of_file> можно с раширением <extension> (расширение(docs, txt)

61  rm -> удаление файла

rm -rf ->Принудительное удаление вместе с содержимоми

5. pwd (print workind directory) -> выводит путь до текущей директории

7. CTRL+l - clear ->  Очистить экран терминал

8. source - текущая директория

9. ls -a -> выводит список как и скрытых так и обычных файлов

10. ls -l ->  выводит список директории и файлов с подробной информацией

11. mv (move) -> перемещает или переименовывает файл или директории в зависимости от указанного второго аргумента
mv <файл _директории> <новое_имя_директории/>

12. cat -> выводит содержимое указанного файла
cat для копирование содержимого файла -> cat (имя файла) -> имя нового файла (cat test.txt > test1.txt)

13. nano -> Открывает графический интерфейс в терминале для редактирование файла
*если нет указанного файла, то создастта файл с таким именем
nano <name_of_file>

14. echo -> команда для быстрой записи данных в файл через терминал
echo 'Текст для записи' > название файла
echo>test.txt'Hello world' -> Запись внутри созданного файла. (При это все старые содержимое очиститься)
echo 'Hello world' -> для создание нового файла и с записью


15. cp - копировать
